Code Explorer
Code Explorer panel will allow a developer to generate the source code for an individual component of one's choice. The Code Explorer in CodeBhagat tool, will provide a list of individual component that you can generate in a tree view form arranged by Technology and/or an Application Layer.

To generate an individual component,

  1. Make sure that Code Explorer is open. If it is not, click the View menu and select Code Explorer.
  2. Select an individual component of your choice and click on Geneate icon in Code Explorer menu bar.
    Note: You may also Double Click on an individual component to generate the source code.
