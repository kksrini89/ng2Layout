Creating New Project

You can create a new projet or open an existing project using CodeBhagat tool.

To create a new project,

1. From the File menu, select New Project.


2. Enter Project Name and Project Description and click on OK button to create a new project.

Once you create a new project, you should review the default settings for various code generation options for the new project. Click here to go to Setting Project Options.
