Database Explorer
Database Explorer panel is just like an Object Explorer, where developers can add/remove a database, browse the database objects and query against its objects. The Database Explorer in CodeBhagat tool, however, is bit advanced that also allows you to generate the code for specific components by selecting a database object and using Context Menu items.

To add a database,

  1. Make sure that Database Explorer is open. If it is not, click the View menu and select Database Explorer.
  2. Click on Add from the menu and select the type of database you want to add.
  3. In the Database Connection dialog box, enter the necessary parameters for database connection based on type of database you are connecting to, and click OK to add the database to the Database Explorer.
    Note: You may click on "Test Connection" button to test your database connections before adding to the Database Explorer.

To delete a database,

  1. Make sure that Database Explorer is open. If it is not, click the View menu and select Database Explorer.
  2. Right Click on the database node you want to delete from the tree view, and select Delete Connection from the context menu OR click on "Delete" menu item.

To generate code from Database Explorer,

  1. Right Click on the database object node (i.e. Table, View, Stored Procedure, or UDF), and select Quick Generate from the context menu.
  2. Then, select the type of code you want to generate from the context menu.
