Setting Database Connections

Setting up database connection is the first step towards generating the code using CodeBhagat tool. Once you have setup the database connection, you can select a specific database object (Table, View, Stored Procedure or UDF) and generate code for all the layers using just one-click action.

You can use CodeBhagat to generate code from SQL Server database (supports SQL Server 2000, 2005 and 2008), SQL Standalone database (.mdf file), Oracle and MS-Access (.mdb and .accdb) databases.

To connect to SQL Server database,


1. From the Database Explorer menu, select Add and click on SQL Server Database menu item.



2. The above selection will open the New SQL Connection dialog box as shown below.



3. You can provide necessary connection information and click OK button to add the connection to the project and Database Explorer window.


To Refresh an existing connection,


1. From the Database Explorer menu, select Database Connection node and click on Refresh menu item.
    This will refresh the selected database connection with latest changes in the Database Explorer window.


To modify an existing connection,


1. From the Database Explorer menu, select Database Connection node and click on Modify Connection menu item.
    This will open an Edit SQL Connection screen (same as New SQL Connection screen) that will allow you to modify the connection properties for the selected database connection.




To delete an existing connection,


1. From the Database Explorer menu, select Database Connection node and click on Delete menu item.
    This will remove the selected database connection from the Database Explorer window.


To Generate a connection string for your web.config file,


1. From the Database Explorer menu, select Database Connection node and click on Generate Connection String menu item.
    This will generate a connection string that can be copied and placed into your web.config file.


To connect to SQL Server Standalone database,


1. From the Database Explorer menu, select Add and click on SQL Server Standalone Database menu item.



2. The above selection will open the New SQL Connection dialog box as shown below.



3. You can provide necessary connection information and click OK button to add the connection to the project and Database Explorer window.


To connect to Oracle database,


1. From the Database Explorer menu, select Add and click on Oracle Database menu item.



2. The above selection will open the New Oracle Connection dialog box as shown below.



3. You can provide necessary connection information and click OK button to add the connection to the project and Database Explorer window.


To connect to MS-Access database,


1. From the Database Explorer menu, select Add and click on MS Access Database menu item.



2. The above selection will open the New MS Access Connection dialog box as shown below.



3. You can provide necessary connection information and click OK button to add the connection to the project and Database Explorer window.
