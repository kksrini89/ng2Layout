Code Generation

CodeBhagat for .NET, allows you to generate code for an individual component of your choice for a selected table/view or all tables/views. This is a unique feature that most of the code generators are missing. This is useful when you are in the middle of development cycle and you want to generate/re-generate an individual component and apply to your projects.

To generate code for an individual component, you have two options:

1. Quick Generate, which allows you to generate the code with just one-click action by seleting a single object (table or view) OR for all tables/views by selecting Tables node.
2. Custom Generate, which allows you to select or enter various parameters for your specific needs before you generate the code.

Quick Generate - Selected Table/View:

You can use Quick Generate menu to generate the code with just one-click action for a single table or view.

To Quick Generate - Selected table/view,

1. Go to Database Explorer window and expand the database nodes.
2. Right click on the database object (Table or View) to open the context menu as shown below and select Quick Generate menu.

3. Click on the type of code you want to generate from the sub menu. For example, Stored Procedures, Triggers, Data Access Layer class etc.

 

Quick Generate - All Tables/Views:

You can use Quick Generate menu to generate the code for all the tables/views with just one-click action. To Quick Generate for all Tables/Views,

1. Go to Database Explorer window and expand the database nodes.
2. Right click on the Tables node to open the context menu as shown below and select Quick Generate menu.

3. Click on the type of code you want to generate from the sub menu. For example, Stored Procedures, Triggers, Data Access Layer class etc.

This will generate the seleted type of code for all the Tables or Views. If you select ALL from the submenu, you can generate entire Module (includes Stored Procedures, Triggers, DAL Class, Properties Class and Web UI Page or Control) for each table under the Tables node.

You also have an option to generate a Website.map file that can be used to create Menus or Breadcrumbs.

Custom Generate:

You can use Custom Generate menu to select or enter various parameters for your specific needs before you generate the code.

To Custom Generate,

1. Go to Database Explorer window and expand the database nodes.
2. Right click on the database object (Table or View) to open the context menu as shown below and select Generate menu.

3. Click on the type of code you want to generate from the sub menu. For example, Stored Procedures, Triggers, Data Access Layer class etc. This will open a popup window where you can select or enter necessary parameters before you generate the code.

This will generate the seleted type of code for the selected Table or View. If you select ALL from the submenu, you can generate entire Module (includes Stored Procedures, Triggers, DAL Class, Properties Class and Web UI Page or Control) for the selected Table or View.
