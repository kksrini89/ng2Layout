Generate Data Access Layer

You can generate Data Access Layer class by selecting a particular Table or View node from the Database Explorer. This can be done either by selecting Quick Generate or Custom Generate context menu.

Quick Generate:

You can generate the data access layer class code with just one-click action. To Quick Generate,

1. Go to Database Explorer window and expand the database nodes.
2. Right click on the database object (Table or View) to open the context menu as shown below.

3.  Select Quick Generate menu and click on Data Access Layer class from the sub menu. This will generate the data access layer class with the following operations.

GetData() To get all records from a table, returns as DataSet.
GetData(filterExpression, sortExpression, pageIndex, rowsPerPage) To get records by page with specified filter criteria, returns as DataSet.
GetData(filterExpression) To get records with specified filter criteria, returns as DataSet.
GetDetails(ID) To get a specific record by id.
Add(...)* To add a record into the table with all the table columns as parameters, returns newly added record ID.
Update(...)* To update a record into the table with all the table columns as parameters, returns true or false.
Add(object)* To insert a record into the table with DataObject as parameter, returns newly added record ID.
Update(object)* To update a record into the table with DataObject as parameter, returns true or false.
Delete(ID)* To delete a record from the table, returns true or false.
Delete(filterExpression)* To delete records from the table with specified criteria, returns number of records deleted successfully.

* These methods/operations will not be generated for a data access layer class when the database View is selected.

Custom Generate:

You can choose to custom generate the data access layer class code that will allow you to select the methods/operations that you want to generate and the name you want to give to the data access layer class. To Custom Generate,

1. Go to Database Explorer window and expand the database nodes.
2. Right click on the database object (Table or View) to open the context menu as shown below.

3. Click on the Data Access Layer Class from the sub menu to generate the code with specific methods/operations.

When you select Data Access Layer Class, this will open the following popup window where you can select specific methods and enter the name of the class before you generate the code.

Note:
1) The code can be generated either in C# or VB.NET based on the default language you selected in the Tools, Options.
2) You can generate the code using SqlClient, OleDbClient or Enterprise Library based on the option you selected in the Tools, Options.
