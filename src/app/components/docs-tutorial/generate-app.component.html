Generate Applications

You can generate applications in any of the following technologies using CodeBhagat for .NET.

  • Angular 2.0 SPA Web Application
  • AngularJS 1.x SPA Web Application
  • ASP.NET MVC Web Application
  • ASP.NET Web Application
  • Web API Application
  • WPF MVVM Application
  • WCF or WCF Restful Service Library
  • Data Access Layer Component

There are 2 ways to generate an application with your choice of technology.

1. From the File menu, select New Application and select the Application Type from the submenu.


2. From Start Page, select the link for the Application Type you want to generate from Generate Application list.

Note: You may also select to re-generate one of the Recently Generated applications.

Once you select the type of application you want to generate, you may enter necessary inputs such as:
  • Application Name
  • Location: where you want to store the generated application
  • Language: C# or VB.NET
  • Visual Studio Version: 2015/2013/2012/2010
  • Provider: SQL or Native Client, OleDb Client or Enterprise Library.
  • Command Type: Inline SQL or Stored Procedure

You may select the Database (from the dropdown) and Database Objects (from the treeview) for which you want to generate the code, and click on Generate button to generate an application.

Once the application is generated, it will be open in your choice of Visual Studio version. All you need to do is Compile and Run your application.

All application generation will use same steps, except couple of parameters based on specific type of application.
