Generate Database Triggers

You can generate database triggers by selecting a particular Table node from the Database Explorer. The database triggers code can log field by field audit trail data into an audit table. This can be done either by selecting Quick Generate or Custom Generate context menu.

Quick Generate:

You can generate the database triggers code with just one-click action. To Quick Generate,

1. Go to Database Explorer window and expand the database nodes.
2. Right click on the database Table node to open the context menu as shown below.

3.  Select Quick Generate menu and click on Triggers from the sub menu. This will generate the database triggers for the following operations.

INSERT Insert trigger will log all the fields and their values into an audit log table, when a record is inserted into a selected table.
UPDATE Update trigger will log all the fields and their values that are changed into an audit log table, when a record is updated into a selected table.
DELETE Delete trigger will log all the fields and their values into audit log table, when a record is deleted from a selected table.

Following naming convetions are used while creating the database triggers.

<ProjectPrefix>_<TriggerPrefix>_<TableName>_<TriggerType>

Where,
<ProjectPrefix> is the prefix you defined for project in the Tools, Options menu under Database Layer. For example, nw for Northwind project.
<TriggerPrefix> is the prefix you defined for the database trigger in the Tools, Options menu under Database Layer.
<TableName> is the table object you selected from the context menu.
<TriggerType> can be Insert, Update or Delete.

For example, If your table name is Customer, the following database triggers will be created.

  • nw_tri_customer_insert
  • nw_tri_customer_update
  • nw_tri_customer_delete

If you want to ignore the ProjecPrefix and TriggerPrefix, you can choose to remove the prefixes from the Tools, Options menu.

Custom Generate:

You can choose to custom generate the database triggers code that will allow you to select the trigger that you want to generate and the names you want to give to the trigger. To Custom Generate,

1. Go to Database Explorer window and expand the database nodes.
2. Right click on the database Table object to open the context menu as shown below.

3. Click on the Triggers to generate specific triggers from the sub menu.

When you select Triggers, this will open the following popup window where you can select specific triggers and define the names of the trigger before you generate the code.
