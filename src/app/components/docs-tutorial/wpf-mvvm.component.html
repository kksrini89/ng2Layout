Generate WPF MVVM Application

You can generate a new WPF MVVM Application using following steps:

1. From the File menu, select New Application --> WPF MVVM Application menu item.


You will see the following dialog box appear on screen:


2. On "Generate New WPF MVVM Application" dialog, Enter/Select the following parameters/options you want to use to generate the application:

  • Application Name
  • Location: where you want to store the generated application
  • Language: C# or VB.NET
  • Provider: SQL or Native Client, OleDb Client or Enterprise Library.
  • Command Type: Inline SQL or Stored Procedure
  • Visual Studio Version: 2015/2013/2012/2010

3. You may select the Database (from the dropdown) and Database Objects (from the treeview) for which you want to generate the code, and click on "Generate" button to generate an application.
Note: This may take between 1 to 3 minutes based on number of Tables/Views you have selected.

Once the application is generated, it will be open in your choice of Visual Studio version. All you need to do is Compile and Run your application.

To Compile, select Build --> Build Solution from the menu item in Visual Studio.
To Run, click on Start button on Toolbar.
