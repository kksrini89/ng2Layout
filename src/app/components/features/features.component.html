Features

CodeBhagat for .NET
Choice of Technology Allows you to generate code for any of the following technologies:
  • Angular 2.0/TypeScript
  • AngularJS 1.x
  • ASP.NET MVC
  • ASP.NET Web Forms
  • Web API Services
  • WCF/WCF RESTful Services
  • WPF using MVVM Pattern
Choice of Language Allows you to generate code either in C# or VB.NET
Choice of Database Allows you to generate code straight from one of the 8 supported databases:
  • SQL Server
  • SQL Azure
  • Oracle
  • MySQL
  • SQL Standalone Databases (*.mdf)
  • SQL Mobile/CE (*.sdf)
  • SQL Express
  • MS-Access
Entire Application vs Individual Component Allows you to generate code for an entire application or individual components during development cycle. So, when the schema of a particular table/view changes or new table is introduced in the database, I can generate an individual component (UI, Business, DAL, Service Layer or Database Layer) and apply the same in my existing application/project.
Choice of Database Provider Allows you to choose the Database Provider: Native Provider (SqlClient for SQL databases, MySqlClient for MySQL, OracleClient for Oracle etc.), OleDb Client as well as Enterprise Library. Many enterprise applications are utilizing Enterprise Library due to dynamic nature of their backend databases.
Customization of Code Templates Easy to change the code generation templates and re-generate an application or individual component. You can make changes to the default code blocks such as Error Handler, Method Signatures etc.
Generate Service Layer Allows you to generate Service Layer/Components:
  • Web API 2.0 Services
  • WCF and WCF RESTful Sevices
  • SOAP/ASMX Web Services
