A Unique, Intelligent and Real Code Generator for .NET!

System Requirements

CodeBhagat for .NET can be installed on any of the operating systems mentioned below. The user's machine must have Microsoft .NET Framework 4.0 or higher and one or more of the database softwares that our product supports (MS SQL Server, MS SQL Express, MS Access and Oracle) in order to generate the code for that database. You will also need the Integrated Development Environment such as Microsoft Visual Studio.NET and .NET SDK in order to modify the code generated by CodeBhagat software.

  • Windows 8 (x86 and x64)
  • Windows 8.1 (x86 and x64)
  • Windows 7 SP1 or later(x86 and x64)
  • Windows XP SP1 or later (x86 and x64)
  • Windows Server 2012 R2 (x64)
  • Windows Server 2012 (x64)
  • Windows Server 2008 R2 SP1 (x64)
